Class: Logger

Logger(name)

new Logger(name)

Logger to log messages to the defined appender.

Default appender is Appender, which is ignoring all messages. Please use setAppender() to set a specific appender (e.g. WindowAppender). use {@see Log4js#getLogger(String)} to get an instance.

Set the date format of logger. Following switches are supported:

  • MM - the month
  • dd - the day of month
  • hh - the hour
  • mm - minutes
  • O - timezone offset
