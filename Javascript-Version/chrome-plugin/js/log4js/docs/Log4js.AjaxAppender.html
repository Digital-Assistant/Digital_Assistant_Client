Class: AjaxAppender

AjaxAppender(logger, loggingUrl)

new AjaxAppender(logger, loggingUrl)

AJAX Appender sending Log4js.LoggingEvents asynchron via XMLHttpRequest to server.
The Log4js.LoggingEvent is POSTed as response content and is formatted by the accociated layout. Default layout is Log4js.XMLLayout. The threshold defines when the logs should be send to the server. By default every event is sent on its own (threshold=1). If it is set to 10, then the events are send in groups of 10 events.
Parameters:
Name Type Description
logger Log4js.Logger log4js instance this appender is attached to
loggingUrl String url where appender will post log messages to
